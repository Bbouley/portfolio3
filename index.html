Study Up

A study application built to help users save tutorials they have found online and share them. Includes authentication using passport. Built with Angular, Node and Express. Also uses Skeleton as a front-end framework in place of Bootstrap. I made use of nested schemas in mongo to help develop a relational database. I used a lot of tests to make sure all my routes on the back end were working properly with my database before starting on the front end.

My first project. Built using Jquery and vanilla Javascript. It is only a front-end project, however I used firebase to set up a database that I could write to.
